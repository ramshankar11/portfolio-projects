# AWS Serverless REST API

This project defines a scalable serverless architecture on AWS using the **Serverless Application Model (SAM)**.

## Architecture
- **API Gateway**: Entry point for REST requests (`GET /users/{id}`).
- **AWS Lambda**: Python function to process logic.
- **Amazon DynamoDB**: NoSQL database for user storage.
- **IAM Roles**: Least-privilege policies automatically generated by SAM.

## Prerequisites
- AWS CLI
- AWS SAM CLI
- Python 3.9

## How to Deploy
1. **Build**:
   ```bash
   sam build
   ```

2. **Deploy**:
   ```bash
   sam deploy --guided
   ```
   Follow the prompts to name your stack (e.g., `serverless-api`) and select a region.

3. **Test**:
   Once deployed, SAM outputs the `ApiUrl`.
   ```bash
   curl https://<api-id>.execute-api.us-east-1.amazonaws.com/Prod/users/123
   ```

## Key Benefits
- **Zero Server Management**: No EC2 instances to patch.
- **Pay-per-use**: Costs ensure only when code runs.
- **Iac**: Entire stack defined in `template.yaml`.
