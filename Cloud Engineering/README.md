# Cloud Engineering Portfolio

A collection of 5 projects demonstrating multi-cloud expertise across **Google Cloud Platform (GCP)**, **Amazon Web Services (AWS)**, and **Microsoft Azure**.

## Projects Overview

| Cloud | Project | Tech Stack | Key Concepts |
|-------|---------|------------|--------------|
| **GCP** | [1. 3-Tier Infrastructure](./GCP_Terraform_Infra) | Terraform, Compute Engine, VPC | Infrastructure as Code (IaC), Networking, Load Balancing |
| **GCP** | [2. Kubernetes Microservices](./GCP_GKE_Microservices) | GKE, Terraform, Helm | Container Orchestration, Scalability, Managed K8s |
| **GCP** | [3. Cloud Run Pipeline](./GCP_CloudRun_Pipeline) | Cloud Run, Python, Docker | Serverless Containers, Event-Driven Architecture |
| **AWS** | [4. Serverless API](./AWS_Serverless_API) | Lambda, API Gateway, SAM | Serverless Functions, NoSQL (DynamoDB), REST APIs |
| **Azure**| [5. VM Configuration](./Azure_Ansible_Config) | Bicep, Ansible, Linux | Cloud Provisioning, Configuration Management, Security |

## How to Use
Each directory contains code and a `README.md` with specific deployment instructions.
**Note**: Most projects require active cloud credentials/CLI tools (`gcloud`, `aws`, `az`) to deploy.
